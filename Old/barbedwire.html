Barbed wire created for the use in my ongoing project “Outnumbered”. There are currently 2 versions a Low and a high poly version.


The high poly version is completely modelled and textured, and looks good when up close.


The low poly version uses a simple set of repeated planes and textures, the textures are just simple diffuse, normals, metallic and opacity maps. This low poly version is better for use in the game as the player is never close enough to see the detail of the high poly one. So we might as well use the low poly one for optimisation. Before the final release of the project I plan to revisit this asset and recreate the low poly one using new unwrap skills and to get a better scaling on the maps.
